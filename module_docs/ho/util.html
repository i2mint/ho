class ho.util.SimpleMappingNamespace[source]

A simple mapping namespace that allows attribute access and dictionary-like access.

>>> s = SimpleMappingNamespace(a=1, b=2)
>>> list(s)
['a', 'b']
>>> s.a
1
>>> s['b']
2
